# Linux System Administration Fundamentals

## Objective
Demonstrate foundational Linux system administration skills including user management, permissions, service access, scheduling, and log analysis.

## Environment
- OS: Ubuntu Linux
- Access: Local terminal / SSH
- Tools: bash, systemctl, journalctl, cron

## Tasks Performed

### User & Group Management
- Created local users and groups
- Assigned group memberships
- Verified permissions and access levels

### File & Permission Management
- Managed ownership using chown
- Applied permissions using chmod
- Tested access control by user and group

### SSH Access & Security
- Verified SSH service status
- Reviewed SSH configuration
- Tested remote login and access restrictions

### Scheduled Tasks
- Created cron jobs for routine maintenance tasks
- Verified execution via logs

### Log Review & Troubleshooting
- Reviewed system logs using journalctl
- Identified authentication and service-related events

## Validation
- Verified user access boundaries
- Confirmed cron job execution
- Tested SSH connectivity and login behavior
